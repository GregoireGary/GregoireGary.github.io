Etape 4 - Professeurs

Vous pouvez soit importer un fichier, soit telecharger un modele base sur votre configuration, soit saisir les professeurs manuellement.

Pour l import, le systeme detecte automatiquement les colonnes: nom du professeur, classes de professeur principal (formats acceptes: 52, 5e2, 5eme2, 5 em2), puis colonnes de classes via les entetes (ex: 6e1, 61, 6eme1, 6 eme 1). L ordre habituel col1 col2 col3... reste prioritaire si coherent.
